# 🏡 European Real Estate Analytics Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard designed to analyze the European real estate market. It focuses on property pricing, rental trends, and the impact of amenities on property value.

---

## 🎯 Business Objectives
- Analyze sales vs rental trends  
- Identify price variations across countries  
- Understand the impact of amenities on property prices  
- Discover high-value and low-cost investment opportunities  
- Track property performance over time  

---

## 🛠️ Tools & Technologies
- Power BI  
- SQL  
- DAX  
- Data Modeling
